Instalasi PTP AP Outdoor

Nopember 2021 Networking

Project Overview

Proyek ini merupakan kegiatan instalasi dan konfigurasi awal server Lenovo ThinkSystem yang dirancang untuk mendukung kebutuhan infrastruktur IT, seperti sistem informasi, database, layanan jaringan internal, maupun virtualisasi. Server ini dipasang di ruang server sekolah/instansi dengan memperhatikan standar keamanan, kestabilan daya, dan efisiensi pendinginan.

Instalasi dilakukan oleh tim teknis dengan prosedur yang memastikan performa optimal dan kompatibilitas dengan sistem yang sudah ada.

Key Features

  • Instalasi fisik server Lenovo ThinkSystem pada rack mount
  • Konfigurasi BIOS dan RAID untuk optimalisasi storage
  • Instalasi sistem operasi server (Windows Server/Linux)
  • Setup jaringan dan IP statis untuk integrasi LAN
  • Pengaturan sistem monitoring suhu dan daya
  • Aktivasi fitur remote management (Lenovo XClarity Controller)
  • Dokumentasi konfigurasi dan struktur jaringan

Technologies Used

Lenovo ThinkSystem RAID Controller & Storage SSD Enterprise Mounting kit dan tiang tower Sistem Operasi: Proxmox IPMI/XClarity untuk remote monitoring UPS (Uninterruptible Power Supply) Rack server dan kabel patch panel

Project Challenges

Tantangan utama dalam proyek ini adalah penyesuaian spesifikasi server dengan kebutuhan sistem dan kapasitas kerja jangka panjang. Untuk mengatasi hal ini, dilakukan analisa kebutuhan resource (CPU, RAM, storage) serta perencanaan skalabilitas.

Selain itu, manajemen suhu dan daya juga menjadi perhatian, mengingat server harus bekerja 24/7 secara stabil. Solusinya adalah dengan memasang server di dalam rack yang dilengkapi sistem pendingin, serta menambahkan UPS untuk mengantisipasi gangguan daya listrik.
